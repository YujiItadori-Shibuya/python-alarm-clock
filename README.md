# python-alarm-clock

A functional GUI-based alarm clock built using Python's `tkinter` and `threading` libraries.

## Features
- Interactive GUI for setting hours, minutes, and seconds.
- Background threading to prevent the UI from freezing.
- Sound alerts using the `winsound` library.

## Requirements
- Python 3.x
- Windows OS (required for `winsound`)
- A `sound.wav` file in the root directory.

## How to Run
1. Clone the repository.
2. Ensure `sound.wav` is in the folder.
3. Run the script: `python alarm_clock.py`
